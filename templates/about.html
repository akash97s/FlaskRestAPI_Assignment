This appication helps in detecting whether an image is Colored or Black & White

Tech stack: Flask + Javascript + Bootstrap


Steps:

1. Go to detect page

2. Choose the image file

3. Click the submit button to get the result


How the application works:

The app has 3 endpoints

1. /Home -> home.html which is the landiing page

2. /detect -> detect.html which is the Rest API endpoint and fallback page

3. /about -> about which is this page which contains the information abou the application


1. The user selects the image

2. The image file is uploaded and saved in the static/uploads folder

3. The image file is fed to a helper function which provides the results

4. The exception cases are handled with the help of flash messages

5. The application is built with bootstrap to support reponsive design

6. The server logs can be checked for project flow
